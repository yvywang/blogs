Console 对象方法

Method Description Demo
assert 该方法接受两个参数 ( 表达式, 字符串 ) 只有当表达式不成立，才会输出打印字符串
clear 清除当前控制台的所有输出，将光标回置到第一行
group 将打印的信息进行分组，可以折叠和展开查看
groupCollapsed 和 group 方法类似，唯一区别就是初始状态是折叠的
table 将复合类型的数据转为表格显示
time 通常用来计时，和 timeEnd 搭配使用
trace 追踪函数的调用过程
count 用于计数，每执行一次 count 所标记的 字符串 都会打印当前执行的次数
